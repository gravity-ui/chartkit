ChartKitWidgetAxis

ChartKitWidgetAxis: { categories?: string[]; grid?: { enabled?: boolean }; labels?: ChartKitWidgetAxisLabels; lineColor?: string; maxPadding?: number; min?: number; ticks?: { pixelInterval?: number }; timestamps?: number[]; title?: { margin?: number; text?: string }; type?: ChartKitWidgetAxisType }

Type declaration

  • optionalcategories?: string[]
  • optionalgrid?: { enabled?: boolean }

    The grid lines settings.

    • optionalenabled?: boolean

      Enable or disable the grid lines.

      Defaults to true.

  • optionallabels?: ChartKitWidgetAxisLabels

    The axis labels show the number or category for each tick.

  • optionallineColor?: string

    The color of the line marking the axis itself.

  • optionalmaxPadding?: number

    Padding of the max value relative to the length of the axis. A padding of 0.05 will make a 100px axis 5px longer.

    Defaults to 0.05 for Y axis and to 0.01 for X axis.

  • optionalmin?: number

    The minimum value of the axis. If undefined the min value is automatically calculate.

  • optionalticks?: { pixelInterval?: number }
    • optionalpixelInterval?: number

      Pixel interval of the tick marks. Not applicable to categorized axis. The specified value is only a hint; the interval between ticks can be greater or less depending on the data.

  • optionaltimestamps?: number[]
  • optionaltitle?: { margin?: number; text?: string }
    • optionalmargin?: number

      The pixel distance between the axis labels or line and the title.

      Defaults to 4 for horizontal axes, 8 for vertical.

    • optionaltext?: string
  • optionaltype?: ChartKitWidgetAxisType

ChartKitWidgetAxisLabels

ChartKitWidgetAxisLabels: { autoRotation?: boolean; dateFormat?: string; enabled?: boolean; margin?: number; numberFormat?: FormatNumberOptions; padding?: number; rotation?: number; style?: Partial<BaseTextStyle> }

Type declaration

  • optionalautoRotation?: boolean

    For horizontal axes, enable label rotation to prevent overlapping labels. If there is enough space, labels are not rotated. As the chart gets narrower, it will start rotating the labels -45 degrees.

  • optionaldateFormat?: string
  • optionalenabled?: boolean

    Enable or disable the axis labels.

  • optionalmargin?: number

    The label's pixel distance from the perimeter of the plot area.

    @default: 10

  • optionalnumberFormat?: FormatNumberOptions
  • optionalpadding?: number

    The pixel padding for axis labels, to ensure white space between them.

    @defaults: 5

  • optionalrotation?: number

    Rotation of the labels in degrees.

    @default: 0

  • optionalstyle?: Partial<BaseTextStyle>
