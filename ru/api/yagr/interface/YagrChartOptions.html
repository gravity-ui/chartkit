Main chart visualization config

Properties

optionalappearance

appearance?: { drawOrder?: DrawKey[]; locale?: Record<string, string> | SupportedLocales; theme?: YagrTheme }

Type declaration

  • optionaldrawOrder?: DrawKey[]

    Order of drawing. Impacts on zIndex of entity. (axes, series) by default

  • optionallocale?: Record<string, string> | SupportedLocales

    Locale

  • optionaltheme?: YagrTheme

    Theme (default: 'light')

optionalselect

select?: { minWidth?: number; zoom?: boolean }

Type declaration

  • optionalminWidth?: number

    Minial width to catch selection

  • optionalzoom?: boolean

    Enable native uPlot zoom (default: true)

optionalseries

series?: SeriesOptions

Common series options, could be overriden by series.

<option>
field

optionalsize

size?: { adaptive?: boolean; height?: number; padding?: [number, number, number, number]; resizeDebounceMs?: number; width?: number }

Type declaration

  • optionaladaptive?: boolean

    Should chart redraw on container resize (default: true)

  • optionalheight?: number

    height (by default: 100% of root)

  • optionalpadding?: [number, number, number, number]

    padding in css px [top, right, bottom, left] (by default: utils.chart.getPaddingByAxes)

  • optionalresizeDebounceMs?: number

    Debounce timer for ResizeObserver to trigger: (default 100 ms)

  • optionalwidth?: number

    width (by default: 100% of root)

optionaltimeMultiplier

timeMultiplier?: 1 | 0.001

1 for milliseconds, 1e-3 for seconds (default: 1)
