# Documentation

### Installation

```bash
npm run docs:deps
```

### Development

Each locale has its own command for starting due to Docusaurus [limitations](https://docusaurus.io/docs/i18n/tutorial#start-your-site).

```bash
# en locale
npm run docs:start
# ru locale
npm run docs:start:ru
```

### Serve

Check built documentation locally.

```bash
npm run docs:serve
```
